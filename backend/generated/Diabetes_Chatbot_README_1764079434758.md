# Diabetes Chatbot
A full-stack health management system for diabetic patients featuring a RAG-based AI chatbot, OCR prescription scanner, and modules for glucose tracking, meals, medications, reminders, and PDF reports.

## Description
The Diabetes Chatbot is a comprehensive health management system designed to assist diabetic patients in managing their condition. The system includes a range of features, such as a chatbot that provides personalized advice and support, a prescription scanner that uses OCR technology to digitize prescriptions, and modules for tracking glucose levels, meals, medications, and reminders. The system also generates PDF reports to help patients and healthcare professionals monitor progress and make informed decisions.

## Tech Stack
The project utilizes a combination of technologies, including:
* Node.js and Express.js for the backend
* TypeScript and React for the frontend
* MongoDB for database management
* Cron jobs for scheduling reminders
* Tailwind CSS for styling
* PostCSS for CSS processing

## Installation
To install the project, follow these steps:
1. Clone the repository to your local machine
2. Navigate to the backend folder and run `npm install` to install dependencies
3. Navigate to the frontend folder and run `npm install` to install dependencies
4. Create a `.env` file in the backend folder and add your MongoDB connection string and other environment variables
5. Start the backend server by running `node server.js`
6. Start the frontend development server by running `npm run dev`

## Usage
To use the Diabetes Chatbot, follow these steps:
1. Open a web browser and navigate to the frontend URL
2. Sign up for an account or log in if you already have one
3. Interact with the chatbot to receive personalized advice and support
4. Use the prescription scanner to digitize your prescriptions
5. Track your glucose levels, meals, medications, and reminders using the various modules
6. Generate PDF reports to monitor your progress and share with healthcare professionals

## Features
The Diabetes Chatbot includes the following features:
* RAG-based AI chatbot for personalized advice and support
* OCR prescription scanner for digitizing prescriptions
* Modules for tracking glucose levels, meals, medications, and reminders
* PDF report generation for monitoring progress and sharing with healthcare professionals
* User authentication and authorization for secure access
* Cron jobs for scheduling reminders

## Folder Structure
The project is organized into the following folders:
* `backend`: contains the server-side code, including routes, controllers, models, and services
* `frontend`: contains the client-side code, including React components, API calls, and styling
* `config`: contains database connection settings and other environment variables
* `models`: contains database schema definitions
* `routes`: contains API route definitions
* `services`: contains business logic and utility functions
* `public`: contains static assets, such as images and fonts
