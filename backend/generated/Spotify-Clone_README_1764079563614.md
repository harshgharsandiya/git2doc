# Spotify-Clone
A music streaming web application, mimicking the functionality of Spotify, built using HTML, CSS, and JavaScript.

## Description
This project is a simplified version of the popular music streaming platform, Spotify. It features a user-friendly interface, allowing users to browse and play music from a predefined list of songs. The application includes basic controls such as play, pause, next, and previous, as well as a progress bar to track the current song's playback.

## Tech Stack
* Frontend: HTML, CSS, JavaScript
* Icons: Font Awesome
* Audio Playback: Native HTML5 Audio

## Installation
To run this project, simply open the `index.html` file in a web browser. No additional installation or setup is required.

## Usage
1. Open the `index.html` file in a web browser.
2. Browse the list of available songs and click the play button to start playback.
3. Use the controls at the bottom of the page to pause, play, skip to the next song, or go back to the previous song.
4. The progress bar will update in real-time to reflect the current song's playback position.

## Features
* User-friendly interface with a list of available songs
* Basic audio controls (play, pause, next, previous)
* Progress bar to track the current song's playback
* Predefined list of songs with corresponding cover art

## Folder Structure
* `covers`: contains cover art images for each song
* `songs`: contains audio files for each song
* `index.html`: the main entry point of the application
* `script.js`: contains JavaScript code for handling audio playback and controls
* `style.css`: contains CSS styles for the application's layout and design
